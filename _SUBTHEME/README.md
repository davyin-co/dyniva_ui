# 前端开发
## **1. npm install 安装前端开发工具**
> cnpm install find have some isuue

## **2. bower install 安装主题依赖包**

> 依赖包里面的内容，只提交需调用的部分，不要全部提交，新的依赖包安装，请save进bower.json

## **3. 开发命令**
```bash
  npm run watch-css
  // 监听 assets/less/* 文件的变动并编译

  npm run watch-js
  // 监听 assets/scripts/index.js 文件和所有关联文件的变动并编译

  npm run watch
	// 同时监听js和css文件的变动并编译

  npm run build-js
  // 压缩编译js文件

  npm run build-css
  // 压缩编译css文件

  npm run build
  // 压缩编译js和css文件

```
